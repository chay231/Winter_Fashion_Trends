# Winter_Fashion_Trends
Winter Fashion Trends — это комплексный аналитический проект, направленный на исследование данных о зимней одежде и аксессуарах для выявления рыночных трендов, потребительских предпочтений и оптимальных ценовых стратегий.
Проект реализует полный цикл анализа данных:
Data Science: обработка, анализ и визуализация в Python
Business Intelligence: интерактивные дашборды в Yandex DataLens
Project Management: версионность и документация на GitHub
Цели проекта:
Выявить наиболее популярные категории зимней одежды
Определить оптимальные ценовые диапазоны

**Стек технологий**
1) Язык программирования: Python 3
2) Библиотеки для анализа: Pandas
3) Визуализация: Matplotlib, Seaborn
4) Инструменты: GoogleColab, GitHub Desktop, Yandex DataLens
